# 🎮 React Tic Tac Toe Game

A simple and interactive **Tic Tac Toe game built with React (Vite setup)**.  
This project demonstrates React state management, game logic, and real-time UI updates with CSS styling.

---

## 🚀 Features

- 🎯 Two-player game (X vs O).
- ⚡ Real-time UI updates using React state.
- 🏆 Automatic win detection (rows, columns, diagonals).
- 🤝 Draw detection.
- 🔄 Reset game functionality.
- 🎨 Smooth hover effects and responsive design.
- 📢 Live status updates (turn / win / draw messages).

---

## 🛠 Tech Stack

- **Frontend:** React.
- **Build Tool:** Vite.
- **Styling:** CSS3.
- **Language:** JavaScript (ES6+).

---

## ⚡ Vite Setup (Project Info)

This project was created using **Vite**, a fast build tool for modern web development.

- 🚀 Supports Hot Module Replacement (HMR) for instant updates.
- ⚡ Fast development and build performance.
- 🔌 Uses official plugins:
  - `@vitejs/plugin-react` (React support).
  - `@vitejs/plugin-react-swc` (faster compilation using SWC).

👉 React Compiler is enabled for optimization. 
👉 ESLint is included for code quality and error checking.

## 🎮 How to Play

1. Player X starts first.
2. Click any empty cell to place your mark.
3. Players take turns (X and O).
4. First player to align 3 marks wins.
5. If all cells are filled → draw.
6. Click **Reset** to restart the game.
